Сергей Добрынин

Журналист, писатель, поэт, публицист, фотограф, сценарист, web-разработчик.

 Посмотреть/скачать резюме

HTML

Свои познания в языке разметки HTML я оцениваю как хорошие – в своей работе использую актуальные семантические теги HTML5, а также применяю протокол OpenGraph для расширения возможностей web-страниц при публикации их в социальных сетях.

CSS

Каскадные таблицы стилей CSS – также уверенно. Использую совре­мен­ные технологии Flexbox и Grid CSS, в случае необходимости могу использовать фреймворки Bootstrap 3 и 4, Foundation и им подобные.

JavaScript

Язык JavaScript знаю на уровне написания простых скриптов и внесения изменений в уже готовые решения. Осваиваю Vue.js.

jQuery

Библиотеку jQuery использую на уровне подключения и иници­ализации с ее помощью сторонних решений (слайдеры, галереи, карты и т.п.), а также изменения и дополнения их базовых настроек.

Node JS

Постепенно внедряю в свою работу возможности по автоматизации повторяю­щихся процессов, предо­ставляемые Node JS – в частности, сборщик проектов Gulp и менеджер пакетов Bower.

PHP

Язык PHP знаю на уровне применения готовых решений и написания несложных скриптов с использованием циклов, условных операторов и т.п.

MySQL

Работаю с SQL базами данных на уровне создания и изменения таблиц и запросов к ним при помощи инструментов администриро­вания баз данных (например, PHP MyAdmin).

Git

Владею базовыми приемами работы с системой контроля версий Git: локальные и удаленные репозитории, GitHub и BitBucket.

WordPress

Систему управления контентом WordPress знаю на уровне уверенного пользователя: установка на хостинг и подключение CMS к базе данных, настройка сайта на WordPress, расширение возможностей с помощью плагинов и администрирование и наполнение сайта.

Sublime Text 3

Ежедневно использую текстовый редактор Sublime Text 3, рас­ширенный за счет плагинов, и Git-клиент Sublime Merge для синхронизации своих рабочих каталогов с удаленным репозиторием.

Figma

Стараюсь использовать «облач­ные» альтернативы Photoshop, поскольку полно­ценной замены ему на Linux до сих пор нет (да, я редко использую Windows).

Ubuntu Linux

Больше десяти лет моей основной операционной системой является Ubuntu Linux, которой я владею на достаточно хорошем уровне.

Sass

Для удобства работы с большими таблицами стилей ввожу в обиход CSS-препроцессор Sass.

Less

То же самое могу сказать и относительно пре­про­цес­сора Less.

Python & Django

Изучаю язык програм­мирования Python; также интересуюсь воз­мож­ностями фрейм­ворка Django для создания сайтов, написан­ного на этом языке. Разрабатываю и обучаю ботов для Telegram.

Ruby & Ruby on Rails

Аналогичным образом обстоит ситуация и с языком Ruby и фрейм­ворком Ruby on Rails.
