Hard skills

HTML

Свои познания в языке разметки HTML я оце­ни­ваю как хорошие – в своей работе использую ак­ту­аль­ные се­ман­ти­чес­кие теги HTML5, а также применяю протокол OpenGraph для расширения воз­мож­но­стей web-страниц при публикации их в со­ци­аль­ных сетях.

CSS

Каскадные таблицы сти­лей CSS – также уверенно. Использую совре­мен­ные технологии Flexbox и Grid CSS, в случае не­об­хо­ди­мос­ти могу использовать фреймворки Bootstrap 3 и 4, Foundation и им по­доб­ные.

JavaScript

Язык JavaScript знаю на уровне написания про­стых скриптов и внесения изменений в уже готовые решения. Осваиваю Vue.js.

jQuery

Библиотеку jQuery ис­поль­зую на уровне под­клю­че­ния и иници­ализации с ее помощью сторонних ре­ше­ний (слайдеры, галереи, карты и т.п.), а также изменения и дополнения их базовых настроек.

Node JS

Постепенно внедряю в свою работу воз­мож­но­сти по автоматизации по­вто­ря­ющих­ся про­цес­сов, пре­до­став­ля­емые Node JS – в частности, сборщик про­ек­тов Gulp и менеджер па­ке­тов Bower.

PHP

Язык PHP знаю на уровне применения готовых ре­ше­ний и написания не­слож­ных скриптов с ис­поль­зо­ва­ни­ем циклов, услов­ных операторов и т.п.

MySQL

Работаю с SQL базами дан­ных на уровне создания и изменения таблиц и за­про­сов к ним при помощи инструментов ад­ми­ни­стри­ро­ва­ния баз данных (например, PHP MyAdmin).

Git

Владею базовыми при­ема­ми работы с системой кон­тро­ля версий Git: ло­каль­ные и удаленные ре­по­зи­то­рии, GitHub и BitBucket.

WordPress

Систему управления кон­тен­том WordPress знаю на уровне уве­рен­ного поль­зо­ва­те­ля: установка на хос­тинг и подключение CMS к базе данных, настройка сайта на WordPress, рас­ши­ре­ние возможностей с помощью плагинов и ад­ми­ни­стри­ро­ва­ние и на­пол­не­ние сайта.

Sublime Text 3

Ежедневно использую тек­сто­вый редактор Sublime Text 3, рас­ширенный за счет плагинов, и Git-кли­ент Sublime Merge для синхронизации своих ра­бо­чих каталогов с уда­лен­ным ре­по­зи­то­ри­ем.

Figma

Стараюсь использовать «облач­ные» аль­тер­на­ти­вы Photoshop, поскольку полно­ценной замены ему на Linux до сих пор нет (да, я редко использую Windows).

Ubuntu Linux

Больше десяти лет моей основной опе­ра­ци­он­ной системой является Ubuntu Linux, которой я владею на достаточно хорошем уров­не.

Sass

Для удобства работы с большими таблицами сти­лей ввожу в обиход CSS-препроцессор Sass.

Less

То же самое могу сказать и от­но­си­тель­но пре­про­цес­сора Less.

Python & Django

Изучаю язык програм­мирования Python; также интересуюсь воз­мож­ностями фрейм­ворка Django для создания сайтов, написан­ного на этом языке. Разрабатываю и обучаю ботов для Telegram.

Ruby & Ruby on Rails

Аналогичным образом обстоит си­ту­ация и с языком Ruby и фрейм­ворком Ruby on Rails.
